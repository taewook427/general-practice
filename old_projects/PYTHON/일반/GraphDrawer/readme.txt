============================================================
GraphDrawer [방정식 그래프 그리개] Old/New Version 사용 설명서
============================================================

2020.07 작성됨, 2021.02 수정됨, 제작 고태욱
New 버전에서 달라진점: 그리기 속도 10.8배 상승, 0으로 나누기 버그 해결
주의사항: Old 버전에서 그래프를 그릴때 0으로 나누는 계산이 있으면 오류가 납니다.('y=1/x' 등)
New 버전은 실행 후 3초 정도 로딩되는 시간이 걸립니다.
거듭제곱과 같이 너무 큰 수가 나오는 수식은 오류가 날 수 있습니다.
0을 음수만큼 거듭제곱 할 수 없습니다.('x^y=y^x'등)
사용할 수 없는 문법의 식을 입력하면 오류가 납니다.('-y+x=0', 'x*y= 60' 등)
실행중인 프로그램을 중간에 닫지 마십시오.

'방정식 그래프 그리개'는 임의의 입력받은 방정식을 그래프로 그려주는 프로그램입니다.
변수로는 x, y를 사용하며 {x|-61<x<61인 정수}, {y|-61<y<61인 정수}의 범위에 대해 그립니다.
그래프에서 분홍색은 정의되지 않은 영역, 회색은 방정식의 영역이 아닌 부분, 파란색이 방정식의 영역입니다.
실행 후 'f(x,y)=/</>g(x,y)' 꼴의 수식을 입력하면 그리기가 시작됩니다.
등식의 경우, 등식 판정 범위를 추가로 입력해야 합니다. 등식 판정 범위는 양의 실수로 입력하십시오.
예를 들어 'y=x'와 '1.0' 을 입력했다면 실제로 그려지는 범위는 {(x,y)|y-1<x<y+1} 입니다.
등식 판정 범위가 클수록 굵게 그려지니, 방정식마다 적절한 판정 범위를 설정하십시오.
식 f와 g는 두 변수 x와 y에 관한 식이여야 합니다. 어떠한 경우에도 띄어쓰기를 하지 마십시오.
사용 가능한 (부)등호는 '=', '<', '>' 입니다. 그래프가 그려지는 범위를 조정하고 싶은 경우,
f(x,y)에서 x와 y부분을 조절하십시오. x축 방향으로 a, y축 방향으로 b만큼 평행이동한 그래프는 f(x-a,y-b)가 됩니다.
같은 원리로 x축 방향으로 m배 확대, y축 방향으로 n배 확대한 그래프는 f(x/m,y/n)가 될 것입니다.
무리수 e와 원주율 pi는 그대로 'e', 'pi'로 쓰면 됩니다. 자동으로 '2.718...', '3.141...'로 변환되어 인식됩니다.
연산자와 함수의 경우, 아래 목록에 나와 있는 경우에만 사용 가능합니다.
계산 우선 순위는 괄호 > 거듭제곱 > 곱셈 / 나눗셈 > 덧셈 / 뺄셈 입니다.

사용 가능한 연산자 목록과 사용 예시:
{'('과 ')'; 안쪽 괄호부터 처리됩니다. 항상 쌍으로 묶어써야 합니다. 함수 표현에도 사용됩니다.
예) '((x/30)^(x/30))>(y/30)^(y/30)', 'y/30=sin(x/10)'}
{'+'와 '-'; 덧셈과 뺄셈의 기호입니다. 모든 연산자는 앞/뒤에 수가 와야 합니다.
예) '0-y=x', '(0-1)*(x+y)>0'}
{'*'과 '/'; 곱셈과 나눗셈의 기호입니다. 미지수나 괄호끼리의 곱이라 하더라도 곱셈을 생략할 수는 없습니다.
0으로 나누는 경우는 정의되지 않습니다. 예) 'y=900/x', 'x*y<1600'}
{'^'; 거듭제곱의 기호입니다. 밑이 0이고 지수가 음수인 경우는 사용할 수 없고, 밑이 음수이고 지수가 정수가 아닌 경우는 정의되지 않습니다.
예) 'y/6>2^(x/20)', '(x^2+y^2-1600)*(x^2+y^2-400)<0'}

사용 가능한 함수 목록과 사용 예시:
{'round()'; 반올림 함수입니다. 소수부분이 0.5이상이면 올림, 미만이면 내림합니다.
예) 'y/20=round(x/20)'}
{'abs()'; 절댓값 함수입니다. 항상 0 혹은 양수를 내보냅니다.
예) 'abs(x)+abs(y)<50'}
{'up()', 'down()'; 올림, 내림 함수입니다.
예) 'y/20=up(x/20)', 'y/20=down(x/20)'}
{'root()'; 제곱근을 반환하는 함수입니다. 음수의 제곱근은 정의되지 않습니다.
세제곱근, 네제곱근 등 일반적인 n제곱근은 거듭제곱을 이용해서 표현해야 합니다.
예) '(y/30)*(y/30)<root(x/30)'}
{'sin()', 'cos()', 'tan()'; 삼각함수입니다. tan((2n+1)/2*pi)는 정의되지 않습니다. 계산은 라디안으로 합니다.
예) 'y/40>sin(x/10)', 'y/40<cos(x/10)', 'y/40=tan(x/10)'}
{'asin()', 'acos()', 'atan()'; 각각 sin, cos, tan의 역함수입니다.
asin과 acos은 {x|-1<=x<=1}에서만 정의됩니다. 계산은 라디안으로 나옵니다.
예) 'y/30>asin(x/50)', y/30=acos(x/50)', 'y/30<atan(x/30)'}
{'ln()', 'log()'; 각각 밑이 무리수 e와 10인 로그입니다. 진수가 0과 음수일때는 정의되지 않습니다.
밑이 다른 로그를 표현하기 위해서는 로그의 밑변환 공식을 이용해야 합니다.
예) 'y/20>ln(x/6)', 'y/20>log(x)/log(7)'}
